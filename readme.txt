Descripción
Esta aplicación es un sistema de gestión de inventarios que permite realizar las siguientes acciones:
. Agregar productos al inventario.
. Modificar productos existentes.
. Eliminar productos del inventario.
. Visualizar el inventario completo.
. Vender productos con manejo de stock y carrito de compras.

Requisitos Previos
1. Tener Instalado Python 3.8 o superior.
2. Instalar las librerías necesarias ejecutando : pip install colorama

Ejecución 
1. Asegúrate de que el archivo inventario.db no exista( se genera automáticamente al ejecutar el programa la primera vez).
2. Ejectuta el programa con: ProyectoIntegradorVivianC.py
3. Sigue las opciones del menú principal para interactuar con el inventario.

Funcionalidades 
1. Agregar producto: Permite registrar un nuevo producto con detalles como nombre, descripción , categoría, cantidad y precio.
2. Modificar producto: Modifica información existente de un producto seleccionado.
3. Elimina producto: Elimina un prodducto del inventarío por su número de producto.
4. Mostar inventario: Muestra todos los productos registrados con detalle y alerta de bajo stock si aplica.
5. Vender producto: Permite seleccionar productos para la venta , reducir su cantidad en stock y calcular el total de la compra.
6. Salir: Finaliza la ejecución del programa.

Notas 
. El programa utiliza una base de datos SQLite (inventario.db) para almacenar datos.
. Los colores en la terminal mejoran la experiencia del usuario, pero son opcionales.
